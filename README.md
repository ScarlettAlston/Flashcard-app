# Flashcard-O-Matic App

The flashcard-o-matic app is an app made for students where they can create, edit, and use flashcard decks and cards for studying purposes

## Screenshots

### Home Page:

<img width="1319" alt="Screen Shot 2022-02-22 at 2 10 41 AM" src="https://user-images.githubusercontent.com/92840656/155112878-07a49d72-a72e-4bd9-b1e3-baadf9cd1ebc.png">

### View:
When user clicks to view a selected deck, it will show the deck's title and description with an option to edit, study or add a card to the deck. Viewing a deck also shows each card within the deck's front and back text with an option to edit or delete each card

<img width="1372" alt="Screen Shot 2022-02-22 at 2 27 56 AM" src="https://user-images.githubusercontent.com/92840656/155118723-5b346bf9-8db1-4675-a27e-3c6c5a059888.png">

### Study:
User can study through the selected deck of flashcards. The deck chosen to study will cycle through cards. Each card will have a 'flip' button that will allow the user to flip each card over to view the back of each card's text. Once the card is flipped, the user can click 'next' to move on to the next card in the deck.

### Create New Deck:
User has the option to create a brand new deck for different courses or topics. When creating a new deck, the deck must have a title and description. They may then fill the deck with flashcards.

### Add Card:
User can add a card to their deck of choosing. They can fill in the front and back side of each card with text.
